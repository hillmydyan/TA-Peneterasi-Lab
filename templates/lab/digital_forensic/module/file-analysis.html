File Analysis

Apa itu File Analysis?

Introduction

File analysis atau analisis file merupakan tahap awal yang krusial dalam Digital Forensic. Analisis file merupakan proses pemeriksaan file digital untuk mengidentifikasi struktur, konten, dan artefak yang relevan dalam investigasi forensik digital.

Artefak dalam Digital Forensic adalah sisa jejak, data, atau informasi digital yang tertinggal dari aktivitas pengguna pada sistem atau perangkat digital.

Lesson Overview

  • File memiliki struktur internal (header, body, footer)
  • Ekstensi file tidak selalu mencerminkan format sebenarnya
  • File dapat dimodifikasi atau dimanipulasi
  • File analysis menjadi dasar analisis forensik lanjutan

Practice Scenario

Langkah 1 - Pemilihan Bukti

Periksa bukti digital dalam directory evidence.

cd evidence/file_analysis
ls -lah

ls -lah: Menampilkan detail file (permission, owner, size, timestamp).

Langkah 2 - Verifikasi Tipe Asli File

Identifikasi apakah ekstensi file sama dengan tipe asli nya.

file *

Mengidentifikasi tipe file sebenarnya berdasarkan signature (magic bytes).

Langkah 3 - Analisis Metadata & Timestamp

Analisis metadata file.

stat {nama_file}
exiftool {nama_file}
Langkah 4 - File Header Inspection

Konfirmasi keaslian dari file bukti digital.

xxd {nama_file} | head -n 20
Langkah 5 - Temukan Data Tersembunyi

Identifikasi file tersembunyi.

binwalk {nama_file}
binwalk -e {nama_file}
