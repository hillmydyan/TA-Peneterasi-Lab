Network Forensics

Apa itu Network Forensics?

Introduction

Network forensic adalah suatu cabang dari digital forensik yang memfokuskan diri pada analisis jaringan digital, termasuk struktur jaringan, protokol, dan metode identifikasi dan ekstraksi informasi. Digunakan untuk merekonstruksi aktivitas pengguna dan mengidentifikasi pola perilaku.

Learning Outcomes

  • Memahami konsep dasar analisis traffic jaringan
  • Mengidentifikasi protokol dan anomali
  • Analisis PCAP

Practice Scenario

Langkah 1 - Traffic Overview

Memahami struktur umum dari lalu lintas jaringan.

cd evidence/network_forensic
tshark -r {nama_file}.pcap
Langkah 2 - Identifikasi Protokol Jaringan

Periksa lalu lintas jaringan untuk statistik protokol.

tshark -r {nama_file}.pcap -q -z io,phs
Langkah 3 - Investigasi DNS

Identifikasi domain queries yang mencurigakan.

tshark -r {nama_file}.pcap -Y dns -T fields -e dns.qry.name
Langkah 4 - Analisis Traffic HTTP

Inspeksi lalu lintas HTTP.

tshark -r {nama_file}.pcap -Y http.request
Langkah 5 - Analisis IP Address

Identifikasi hubungan IP yang mencurigakan.

tshark -r {nama_file}.pcap -T fields -e ip.src -e ip.dst
